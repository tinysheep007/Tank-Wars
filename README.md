# Tank-Wars

This is a for fun project to practice JAVA SE.

Press 1 or 2 and then Enter to select mode
Press 3 to pause the game

For Player 1:
Use WASD to move and SPACE to shoot

For Player 2:
User IKJL to move and Y to shoot
